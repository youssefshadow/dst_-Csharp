# dst_-Csharp

Lancer un exercice

# 1. Menu principal
Après avoir lancé le programme, vous verrez un menu avec plusieurs exercices disponibles.
Vous pourrez choisir un exercice en entrant son numéro correspondant :

Sélectionnez un exercice à lancer :
1. Jeu de cartes
2. Exercice des chiffres
3. Exercice du carré évidé
4. Exercice des chaînes de caractères
0. Quitter

# 2. Choisir un exercice

Entrez le numéro de l'exercice que vous souhaitez exécuter


# 3. Quitter le programme

Vous pouvez quitter le programme à tout moment en choisissant l'option "0" dans le menu.

# 4. Continuer ou quitter après chaque exercice

Après chaque exercice, vous serez invité à continuer ou à quitter le programme :

Voulez-vous continuer ? (O/N)

Entrez "O" pour continuer avec un autre exercice, ou "N" pour quitter le programme.
